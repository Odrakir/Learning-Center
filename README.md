# Learning-Center

A personal learning center, powered by the [`teach`](.claude/skills/teach/SKILL.md) skill.

## How it works

Each topic lives in its **own folder** at the repo root — one self-contained
teaching workspace per topic (the skill's rule: *one mission per workspace*).
A topic folder contains:

- `MISSION.md` — why I'm learning this; grounds every lesson
- `RESOURCES.md` — curated high-trust sources (knowledge) and communities (wisdom)
- `lessons/` — self-contained HTML lessons, numbered `0001-…`
- `reference/` — reusable cheat sheets / quick references
- `learning-records/` — what I've actually learned (created lazily)
- `NOTES.md` — teaching preferences and working notes

## Topics

- [`engineering-management/`](engineering-management/) — becoming a good
  engineering manager (new EM, first year).

## Using the skill

Run `/teach <topic>` and work inside that topic's folder. To start a new
topic, create a new sibling folder and treat it as a fresh workspace.
